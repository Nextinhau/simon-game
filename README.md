# Simon Game

## 描述

Simon Game，又称西蒙游戏，一款益智小游戏

## 游戏规则

玩家点击空格键开始游戏，玩家需要记住由AI控制嘅，唔同颜色嘅灯嘅亮灯顺序，然后根据亮灯顺序依次选择四个方向键按下，如果次序与AI给予的次序相同，则游戏继续并增加难度，否则游戏结束，重置游戏

## 用法

下载所有文件，解压后打开`index.html`即可

## 技术栈

- HTML
- CSS
- JS
- SCSS
- JQuery
